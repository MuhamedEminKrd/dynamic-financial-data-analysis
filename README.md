# 📊 Dynamic Financial Data Analysis Panel (MAANG 2000-2023)

This project provides a deep-dive analysis of MAANG (Meta, Amazon, Apple, Netflix, Google) stock data over a 23-year period. It focuses on **data cleaning**, **risk assessment**, and **trend visualization** using industry-standard Python libraries.

## 🚀 Key Features & Engineering Steps

As a Software Engineering student, I followed a rigorous data pipeline:

* **Data Preprocessing & Cleaning:** * Identified and handled null values using `dropna()`.
    * Removed duplicate records to ensure data integrity.
    * Converted date strings to `datetime` objects for accurate time-series indexing.
* **Statistical Analysis (NumPy & Pandas):**
    * **Volatility Calculation:** Measured investment risk using standard deviation of daily returns.
    * **Moving Averages (SMA):** Calculated 50-day and 200-day averages to identify market trends.
* **Comparative Visualization:** * Created interactive time-series plots for price comparison.
    * Developed a **Correlation Heatmap** to analyze market dependencies between tech giants.

## 📈 Analysis Insights
Through this analysis, the system identifies:
1. **Risk Levels:** Which stocks exhibit higher volatility (e.g., Netflix vs. Apple).
2. **Market Correlation:** How closely Microsoft and Apple move together in the long term.
